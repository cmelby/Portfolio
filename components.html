Data Structures

Article Summary: What is Stack? “Stack is an abstract data type with a bounded(predefined) capacity”. In other words its a simple data structure that allows adding and removing elements in a specific order based on how that data is organized. Every time an element is added, it gets placed at the top of the stack. The only element that can be removed is this top element positioned at the top of the stack.
